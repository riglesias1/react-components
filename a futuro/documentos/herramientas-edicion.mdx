## Utilidades basicas:

Si se desea cambiar la paleta de colores utilizada en la documentación se puede hacer desde el **main.css**, entrando desde aquí:

<VSRemote
  title="Server l01"
  description="Conf. visual"
  host="l01.leafnoise.io"
  path="/var/sources/mteyss/documentacion/academy/src/components"
  file="main.css"
/>

Los iconos utilizados se encuentran facilmente para su uso desde la pagina:

<RoundedButton
  title="React Icons"
  route="https://react-icons.github.io/react-icons"
  icon={<FaReact />}
  color="#53C0DD"
/>
<br />

Hay componentes normalizados para cada situación.

- En caso de botones de descarga se utiliza el componente **Slider**
- Para links externos se utiliza **RoundedButton**
- Al tener listas largas de texto es conveniente usar **VerticalTab**
- En el inicio de cada unidad se utiliza **Title**
- Si se necesita renderizar video o imagen estan los componentes **ImageIcon** o **Video**
